---

### Why I built this
Most financial tools online are cluttered with aggressive ads, have outdated designs, or split essential features across multiple pages. I wanted to build a suite that is **fast**, **modern**, **beginner-friendly**, and keeps all calculations local in the browser.

## Key Features
* **100% Local:** All calculations run in the browser. No data is sent to any server.
* **Ad-Free:** No ads, no tracking, no distractions.
* **8 Tools:** A growing suite of professional-grade financial calculators.
* **Responsive:** Fully optimized for both desktop and mobile.
* **Open Source:** Free to use, fork, and contribute to.

## Tech Stack
* **Frontend:** Vanilla HTML5, CSS3, and JavaScript (ES6+). [![Frontend](https://img.shields.io/badge/Frontend-Vanilla%20HTML5%20%2B%20CSS3%20%2B%20JavaScript-yellow)](https://developer.mozilla.org/en-US/docs/Web/HTML)
* **Hosting:** Cloudflare Pages. [![Hosting](https://img.shields.io/badge/Hosting-Cloudflare%20Pages-orange)](https://pages.cloudflare.com/)
* **Design:** Minimalist modern UI. [![Design](https://img.shields.io/badge/Design-Minimalist%20Modern%20UI-green)](https://developer.mozilla.org/en-US/docs/Web/CSS)

## How to use
1. **Online:** Visit the [Live Demo](https://finance-calculators-financial-tools.pages.dev).
2. **Locally:**
   * Clone this repository: `git clone https://github.com/robajzsek-a11y/Finance-Calculators-Financial-Tools.git`
